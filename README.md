# 🎓 AI Learning Buddy Deepa 

A smart web application powered by Streamlit and Google's Gemini-2.5-Flash model. This app functions as a personalized learning assistant designed to break down topics, generate real-world examples, and test knowledge through custom quizzes.

## 🚀 Features
* **Explain Concept:** Simplifies complex topics for beginners.
* **Real-Life Example:** Connects concepts to intuitive analogies.
* **Generate Quiz:** Automatically creates 5 custom multiple-choice questions to test your understanding.
* **Ask Anything:** A free-form conversational prompt for any general learning question.

## 🛠️ Built With
* Python
* Streamlit
* Google Generative AI (Gemini API)
